# National Education Plans

This GitBook site brings together write-ups from our analysis of UNESCO Planipolis education plans, with a focus on whether governments mention evidence-backed education "smart buys" and what else they prioritize.

The site is now organized as plain Markdown rather than Quarto. The GitBook pages are the editable source for the site, while the underlying analysis code, Quarto files, and raw outputs remain outside this repo.

## What You Will Find Here

- a main findings page on government plans and smart buys
- a French-language extension of the broad search
- supporting charts for structured pedagogy and targeted instruction
- a plain-English methodology overview
- detailed notes on the RAG workflow, validation checks, and exact search logic

## Repo Structure

- page content lives in top-level Markdown files
- navigation is controlled through `SUMMARY.md`
- figures used by the pages live in `assets/figures/`

If you want the quickest route into the substance, start with [Introduction](introduction.md) and [Findings overview](findings.md).
